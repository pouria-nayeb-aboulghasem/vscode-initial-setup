### Github

- install git
- Open VSCode
- Press: CTRL + SHIFT + P
- Type: Github: Sign In
- Sign in to your github account from browser
- Return to VSCode
- Initial git configuration

```
    git config --global user.name "your name"
    git config --global user.email "your email"
```

- git commands

```
    git init
    git add .
    git commit -m "description"
```
